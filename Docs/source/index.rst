Tutorial Overview
-----------------
The Tutorial Overview provides some background to this tutorial and links to related resources. 

https://data-vis-with-python.readthedocs.io/en/main/overview.html

Lesson 1
--------
Lesson 1 provides an introduction to plotting with matplotlib and Bokeh using a weather data set containing data for temperature, humidity, wind speed and wind direction. 

https://data-vis-with-python.readthedocs.io/en/main/lesson1.html

Lesson 2
--------
Lesson 2 uses tweet timeline data to show how to use matplotlib for overlaid plots and Bokeh for additional interactivity and metadata display.

https://data-vis-with-python.readthedocs.io/en/main/lesson2.html

Lesson 3
--------
Lesson 3 demonstrates ways in which Seaborn can enhance the visualization of statistical data using the Lahman Baseball Databank.

https://data-vis-with-python.readthedocs.io/en/main/lesson3.html

Lesson 4
--------
Lesson 4 is a short lesson on using networkx and Bokeh for graph visualization. 

https://data-vis-with-python.readthedocs.io/en/main/lesson4.html

Lesson 5
--------
Lesson 5 shows how to apply a Seaborn heatmap to spatiotemporal weather data.

https://data-vis-with-python.readthedocs.io/en/main/lesson5.html
